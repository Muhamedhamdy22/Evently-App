# 🗓️ Evently App

A modern event management mobile application built with Flutter.  
Evently helps users discover, create, and manage events with a clean and intuitive UI.

---

## ✨ Features

- 🔐 User Authentication (Login / Register) via Firebase
- 📅 Browse and manage events
- 🌙 Dark & Light mode support
- 🌍 Multi-language support (Arabic / English)
- 🔥 Real-time data with Firebase Firestore
- 📱 Responsive UI for all screen sizes

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Flutter | UI Framework |
| Dart | Programming Language |
| Firebase Auth | User Authentication |
| Firebase Firestore | Real-time Database |
| Cubit (Bloc) | State Management |

---

## 📂 Project Structure
```
lib/
├── core/         # Constants, themes, utilities
├── features/     # App features (auth, events, etc.)
└── main.dart     # Entry point
```

## 🚀 Getting Started

1. Clone the repo:
```bash
git clone https://github.com/Muhamedhamdy22/Evently-App.git
```
2. Install dependencies:
```bash
flutter pub get
```
3. Run the app:
```bash
flutter run
```
